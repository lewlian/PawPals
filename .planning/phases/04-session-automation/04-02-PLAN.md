<task type="auto">
  <name>Task 1: Create session expiry background job</name>
  <files>src/jobs/sessionExpiry.ts</files>
  <action>
Create new directory src/jobs/ and file src/jobs/sessionExpiry.ts:

1. Imports:
   - bot from '../bot/index.js'
   - Markup from 'telegraf'
   - getSessionsNeedingReminder, getExpiredSessions, expireSessions from '../db/repositories/sessionRepository.js'

2. Constants:
   - POLL_INTERVAL_MS = 30 * 1000 (30 seconds)
   - sentReminders = new Set<number>() for deduplication

3. State:
   - let intervalId: NodeJS.Timeout | null = null

4. Helper function formatReminderMessage(session):
   - Calculate minutesLeft: Math.max(1, Math.round((expiresAt - Date.now()) / 60000))
   - Return casual message per CONTEXT.md: "Hey! Your session at {locationName} ends in {minutesLeft} min(s).\n\nDog(s): {dogNames}\n\nWant to stay longer?"

5. Helper function formatExpiryMessage(session):
   - Calculate durationMinutes from checkedInAt to expiresAt
   - Return: "Your session at {locationName} has ended.\n\nDog(s): {dogNames}\nDuration: {duration} minutes\n\nSee you next time!"

6. Helper function buildExtendKeyboard(sessionId):
   - Return Markup.inlineKeyboard with two rows:
     - Row 1: ['15 min', '30 min', '60 min'] buttons with callback data extend_{sessionId}_{15|30|60}
     - Row 2: ['Checkout now'] button with callback data checkout_{sessionId}

7. async function sendExpiryReminder(session):
   - Build message and keyboard
   - Try bot.telegram.sendMessage(session.telegramId, message, keyboard)
   - Catch and log errors (user may have blocked bot)

8. async function sendExpiryNotification(session):
   - Build expiry message (no keyboard)
   - Try bot.telegram.sendMessage(session.telegramId, message)
   - Catch and log errors

9. export async function processSessionExpiry():
   - Try block:
     - Get sessions needing reminder
     - For each: skip if in sentReminders Set, else sendExpiryReminder and add to Set
     - Get expired sessions
     - For each: sendExpiryNotification, then call expireSessions([session.id])
     - Delete from sentReminders after expiry processed
   - Catch: log error and continue (job should not crash)

10. export function startSessionExpiryJob():
    - Guard: if intervalId exists, warn and return
    - Log "Starting session expiry job (30s interval)"
    - Run processSessionExpiry() immediately for catch-up
    - Set intervalId = setInterval(processSessionExpiry, POLL_INTERVAL_MS)

11. export function stopSessionExpiryJob():
    - If intervalId: clearInterval, set to null, log stopped

12. export function clearReminderTracking(sessionId: number):
    - sentReminders.delete(sessionId)
    - Used when session is extended to allow new reminder
