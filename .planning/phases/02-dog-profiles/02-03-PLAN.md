       This scene handles editing individual fields. It's entered with the dog ID and field to edit stored in scene state.

       **Scene state:**
       ```typescript
       interface EditWizardState {
         dogId: number;
         field: 'name' | 'size' | 'breed' | 'age';
         currentValue: string;
       }
       ```

       **Entry step:**
       - Show current value and prompt for new value
       - For name/age: "Current name: Buddy\n\nEnter new name:"
       - For size: Show inline keyboard with size options
       - For breed: Show "Type to search or browse breeds" with browse button

       **Value step:**
       - Validate input same as creation wizard
       - Call updateDog(dogId, { [field]: newValue })
       - Show success: "{Field} updated! Buddy is now {newValue}"
       - Return to dog detail view (by triggering view_dog callback)
       - Exit scene

       **Action handlers:**
       - size_(.+): Update size field
       - breed_browse_(\d+): Show breed pagination
       - breed_select_(.+): Update breed field

       **Cancel handler:**
       - /cancel exits without saving

    2. Update `src/bot/index.ts`:

       a. Register edit scene with Stage:
          ```typescript
          import { editDogProfileWizard } from './scenes/editDogProfile.js';

          const stage = new Scenes.Stage<BotContext>([
            createDogProfileWizard,
            editDogProfileWizard
          ]);
          ```

       b. Add action handlers for edit buttons:
          ```typescript
          bot.action(/edit_dog_(name|size|breed|age)_(\d+)/, async (ctx) => {
            await ctx.answerCbQuery();
            const field = ctx.match[1];
            const dogId = parseInt(ctx.match[2]);
            // Store in scene state and enter edit wizard
            ctx.scene.enter('edit-dog-profile', { dogId, field });
          });
          ```

       c. Add delete handler with confirmation:
          ```typescript
          bot.action(/delete_dog_(\d+)/, async (ctx) => {
            await ctx.answerCbQuery();
            const dogId = parseInt(ctx.match[1]);
            const dog = await findDogById(dogId);

            await ctx.editMessageText(
              `Are you sure you want to delete ${dog.name}?\n\nThis cannot be undone.`,
              Markup.inlineKeyboard([
                [Markup.button.callback('Yes, delete', `confirm_delete_${dogId}`)],
                [Markup.button.callback('No, keep', `view_dog_${dogId}`)]
              ])
            );
          });

          bot.action(/confirm_delete_(\d+)/, async (ctx) => {
            await ctx.answerCbQuery();
            const dogId = parseInt(ctx.match[1]);
            const dog = await findDogById(dogId);
            await deleteDog(dogId);

            await ctx.editMessageText(`${dog.name} has been removed from your profiles.`);
            // Could auto-show profile list after brief delay
          });
          ```

       d. Add profile_list action to return to dog list:
          ```typescript
          bot.action('profile_list', async (ctx) => {
            await ctx.answerCbQuery();
            // Reuse profile handler logic
            await profileHandler(ctx);
          });
          ```
  </action>
